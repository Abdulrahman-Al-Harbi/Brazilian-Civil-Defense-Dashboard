# Brazilian Civil Defense Dashboard

## Overview

This project presents an interactive Power BI dashboard for analyzing Civil Defense incidents in Brazil using a publicly available dataset from Kaggle.

The dashboard provides insights into complaint trends, economic losses, environmental impacts, infrastructure damage, and regional distributions through interactive visualizations.

---

## Project Objectives

- Analyze Civil Defense incidents across Brazil.
- Monitor complaint trends over time.
- Compare incidents across states and cities.
- Analyze economic losses.
- Explore environmental impacts.
- Visualize infrastructure damage.
- Build an interactive dashboard to support data-driven decision-making.

---

## Dashboard Features

The dashboard includes:

- Total complaints by year
- Economic loss analysis
- Complaint distribution by city
- Complaint status over time
- Environmental issue analysis
- Infrastructure damage analysis
- Interactive filters for:
  - Complaint Type
  - State
  - City
